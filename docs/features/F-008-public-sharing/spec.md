# F-008 — Public Sharing (stub — blocked on backend)

Shareable resume links with on/off toggle and optional password. **Impossible in the
current client-only architecture** — requires hosting, storage, and link/password
management (per ADR-001 the backend milestone is deliberately deferred).

*Stub — do not spec until the backend milestone is scheduled. When it is, this feature
should drive the backend's minimal shape (static hosted render vs. full API) rather
than inheriting the whole SSOT Part 2 schema up front.*
